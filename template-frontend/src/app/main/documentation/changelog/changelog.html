New
  • Boxed layout
  • Angular Material Datepicker and its component page
  • Angular Material Swipe Element, Whiteframe and Menu Bar component pages
  • Language switcher on toolbar
  • Splash screen
  • Imported angular-material.scss to give access to its variables from scss files
  • material-color($colorName, $colorHue) function to get hex values of the Material Colors and their hues in scss.
    Example:
    • color: material-color('red');
    • background-color: material-color('light-blue', '200');
  • Material Design colors and their hues as classes to set different colors on elements:
    Examples:
    • Following classes will set background-color of the element:
      • class="red-bg"
      • class="blue-400-bg"
    • Following classes will set color of the element:
      • class="light-blue-bg"
      • class="light-blue-A100-bg"
    • Following classes will set border-color of the element:
      • class="indigo-border"
      • class="pink-A100-border"
    • Following classes will set border-top-color of the element:
      • class="amber-border-top"
      • class="green-400-border-top"
    • Following classes will set border-right-color of the element:
      • class="amber-border-right"
      • class="green-400-border-right"
    • Following classes will set border-bottom-color of the element:
      • class="amber-border-bottom"
      • class="green-400border-bottom"
    • Following classes will set border-right-color of the element:
      • class="amber-border-right"
      • class="green-400-border-right"
  • Material Colors page for showing Material color classes
