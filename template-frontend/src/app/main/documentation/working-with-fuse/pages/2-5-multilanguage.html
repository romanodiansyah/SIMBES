Working with Fuse Fuse
Multilanguage
angular-translate

Fuse uses angular-translate module to manage multiple languages. We already made some configurations and prepared the directory structure for you to quickly start working with it.

Important Notes

Before you start working on the multiple languages, there are couple things you need to know about angular-translate module. Even though the module itself is out of scope of this documentation, we will talk about couple things to make your life easier.


angular-translate module will NOT work with missing language files. For example, if you add a new language (let's say Portuguese), you have to create pt.json files for every module you have! If angular-translate module is missing even one translation file, it won't activate the language. There are ways to make the module work even with missing files, but you need to check out the official documentation to learn more about it.
