Third Party Libraries

Fuse uses Bower package manager to manage third-party libraries. Combined with the generator, you can add, remove or update third-party libraries in a matter of seconds.

Adding a Third-Party Library

To add a third party library to your application, open your favorite console application (Terminal, Command Prompt etc.), navigate to your work folder and run the following command by replacing the PACKAGE_NAME with the actual name of the package you want to uninstall: bower install PACKAGE_NAME --save

If you are installing an AngularJS module, then you have to add it as a dependency to your application. You can do that either in /app/core/core.module.js file or in /app/index.module.js file.

Removing a Third-Party Library

To remove a third party library from your application, open your favorite console application (Terminal, Command Prompt etc.), navigate to your work folder and run the following command by replacing the PACKAGE_NAME with the actual name of the package you want to install: bower uninstall PACKAGE_NAME --save

If you are uninstalling an AngularJS module, make sure you remove it from your dependencies lists as well. You can check both /app/core/core.module.js file and /app/index.module.js file to make sure you remove the dependency injections.

Note: Other than adding or removing package names to or from the dependency lists, you shouldn't need to anything else to include the actual library into your application. Just reload your server and everything will be injected automatically.

Note: Sometimes, bower packages might have dependencies to other packages. Normally, uninstalling a package will remove its dependencies only if those dependencies are not in use. If you find yourself with lots of unnecessary packages in your bower_components folder, running bower prune command will remove all the unnecessary packages. This should only uninstall the packages that are not in use, but you should still be careful. Bower will output the names of the removed modules to the console, so pay attention when you run the above command and make sure it didn't remove any important packages.
